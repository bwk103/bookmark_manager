# User Stories
```
As a time-pressed user
So that I can quickly go to web sites I regularly visit
I would like to see a list of links on the homepage

As a User
So that I can select my bookmarks
I want to be able to see them on the page.

As a User
So that I can expand my collection of bookmarks
I want to be able to add new bookmarks.

As a User
So that I can properly organise my bookmarks
I want to be able to add tags to bookmarks.

As a User
So that I can find relevant bookmarks quickly
I want to be able to filter bookmarks by tag.
